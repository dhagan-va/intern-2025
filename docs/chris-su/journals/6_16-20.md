### UI OPTIONS
**Tkinter**

Built in, lightweight GUI tookit (bundled with Python)

Backend wraps Tcl/Tk, cross-platform GUI tookit

Pros:
- No dependencies
- Simple API
- Well documented

Cons:
- Outdated visuals
- Clunky styling/layout
- Limited widgets

**PyQt / PySide**

Python bindings for C++ Qt framework

Pros:
- Modern/polished
- Rich feature set 
- Qt designer (drag and drop)

Cons:
- Larger learning curve
- More boilerplate
- Large app bundles

**Web UI with backend + browser**

GUI with HTML/CSS/JS, run locally or embed in desktop

Pros:
- Most flexible
- Responsive, mobile friendly
- Easy to scale

Cons:
- Complex stack
- Potential latency
- Web dev knowledge
