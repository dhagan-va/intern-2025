### Project Overview Notes
- **X12** is a EDI **(Electronic Data Interchange)** standard.
- Provides specification for claims data.
  - Providers submit claims to insurance.
- Veteran's Affairs is largest healthcare provider in the US
  - Has the oldest EHR (Electronic Health Record) in the US
  - 10 million veterans + families, dependents, etc.
  -  Have option of VA facilities or "network" for care.
  -  If providers don't get paid, they stop being in network and veterans have no care.
-  **EDI 270**: Requests information from insurance about coverage.
-  **EDI 271**: Response to inform provider about eligibilty, coverage, benefits.
-  **Goal**: Create a client for the above purpose.
   -  Capable of generating 270 file.
   -  Test client with Tkinter UI
   -  TLS support
   -  Configurable and multithreaded
      -  Adjust request per seconds, concurrent connections.
      -  UI should respond to these changes.
   -  Graph responses
